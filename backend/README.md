## Rundown of Backend

### API Routes (currently in main.py)

- FastAPI routes are denoted by decorators like `@app.get("/")`, followed by
  their function
- Documentation for API routes are generated by FastAPI at either /docs or
  /redoc (e.g. localhost:8000/docs)

### Schemas (currently in schemas.py)

- Backend schemas enforce certain data layout and constraints, so that only
  specified data should be expected in API interactions
- Pydantic is used to provide type hints, as well as constraints such as min/max
  length of a string attribute
- After specifying the `response_model` attribute inside of an FastAPI route,
  FastAPI will handle validation of data for that API route

### Models

- Will provide interaction between Python and the MySQL database using
  SQLAlchemy
